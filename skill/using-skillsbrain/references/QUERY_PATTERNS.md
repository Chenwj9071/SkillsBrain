# 查询模式

## 核心规则

优先使用“动作 + 对象或文件类型”的查询词。

示例：

- `提取 PDF 表格`
- `编辑 Excel 公式`
- `创建 Word 模板`
- `裁剪图片尺寸`

避免只写模糊目标：

- `处理文件`
- `帮我改一下`
- `做个数据操作`

## 多次改写查询

同一任务第一次没有命中时，换一种表达再查，不要只试一次。

可优先按这三类改写：

1. 动作改写：`提取` / `解析` / `导出`
2. 对象改写：`PDF 表格` / `表格数据`
3. 场景改写：`批量处理 Excel` / `Excel 数据清洗`

## 结果判断

- `score >= 0.80`：通常可直接阅读并使用该技能
- `0.70 <= score < 0.80`：大概率相关，建议继续看 `SKILL.md`
- `0.65 <= score < 0.70`：弱相关，建议改写查询词再试
- `< 0.65`：默认不会返回

## 常见任务示例

- 文件处理：`提取 PDF 表格`、`合并 PDF`、`创建 Word 报告`
- 表格处理：`编辑 Excel 单元格`、`批量处理 Excel 数据`
- 格式转换：`JSON 转 CSV`、`图片格式转换`
- 自动化：`批量重命名文件`、`监控文件变化`
